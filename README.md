# Leaf & Lore — Frontend (Next.js)

Next.js 16 frontend for the Leaf & Lore bookshop.

## Setup
1. Install: `npm install`
2. Set NEXT_PUBLIC_API_URL in .env.local to your backend URL
3. Dev: `npm run dev` (runs on port 3000)

## Connect to backend
- Local: set NEXT_PUBLIC_API_URL=http://localhost:4000 in .env.local
- Production (Vercel): set NEXT_PUBLIC_API_URL=https://your-backend.railway.app in Vercel env vars
